MBSE BBS: Which Distribution To Use?

Which distribution to use?

MBSE currently supports Slackware, Debian, Ubuntu, Devuan, and Mageia Linux. MBSE should compile and run under *BSD but it has not been tested in many years.

MBSE should compile and run with little to no modifications on nearly any Linux or *BSD distribution.

If you get MBSE working under a Linux or BSD distribution that is not listed here, please contact us and let us know how you did it! We will document it and, if needed, make changes to allow your distro to compile correctly.
